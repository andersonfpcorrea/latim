# Dativo

1. Interssado na ação:

   - Quid _mihi_ Celsus agit?
   - Quid _mihi_ et tibi, Filii Dei?
   - Fabilam _mihi_ narra

2. Ponto de vista:

   - _Mihi_ es nihil

3. Possessivo:

   - Nomem _mihi_ est Gaius
   - Pecunia multa est _illi_

4. Agente da passiva:

   - Heac _mihi_ dicta sunt

5. Função ou utilidade:

   - _Voluptati_ sunt _ei_ - Sou para o prazer (sirvo para o prazer) para ele (do ponto de vista dele)
   - _Odio_ est _Romanis_ - Ele é ódio (serve para ser odiado) para os romanos (do ponto de vista dos romanos)
